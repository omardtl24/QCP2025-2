
# Qiskit Fest Hackathon

This folder contains Jupyter notebook solutions for the Qiskit Fest Hackathon problems.

**Environment required**

The Python environment defined in `requirements.txt` is required to reproduce the notebooks and run the Qiskit examples.

Quick setup:

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install --upgrade pip
pip install -r requirements.txt
```

Run notebooks with JupyterLab or Jupyter Notebook.
